# Starlink-Constellation-Visualization
This repository presents a Starlink satellite constellation visualization developed using html.

# Features
1. Visualize thousands of Starlink satellites around the Earth.  
2. Select a predefined city or enter custom latitude and longitude coordinates.  
3. Simulate how a user terminal's phased-array antenna connects to the most suitable satellite.  
4. Visualize beam switching as satellites move across the sky.


# Usage Guide
1. Download **Open_in_browser.html** and open it in your web broswer.  
2. Download **Starlink_tle.txt**.  
3. Drag **Starlink_tle.txt** into the web page.  
4. The visualization will load automatically.
