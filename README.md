# Starlink-Constellation-Visualization
This repository displays my work on simple satellite simulation

Usage Guide: Download 'Open_in_browser.html' file and click it to open.
             Download 'Starlink_tle.txt' file and drag it in your browser. You will see the Earth and thousands of Starlink satellites.
             You can chose default cities of enter any lat&lon coordinates to see how terminal phased array antenna links to one of the satellites and beam switches between them.
             Or You can also click 'Starlink Constellation...mp4' file to watch a brisk presentation of this simulation.
